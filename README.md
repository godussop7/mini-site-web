# Mon Mini Site Web

Mini site réalisé dans le cadre du TP Git & GitHub .

## Description

Ce projet est un site web statique simple comprenant :
- Une page HTML structurée (header, sections, footer)
- Une feuille de style CSS (mise en page responsive)
- Ce fichier README

## Structure du projet

```
mon-site/
├── index.html  
├── style.css   
└── README.md   
```

## Technologies utilisées

- HTML5
- CSS3
- Git & GitHub

## Comment lancer le projet

1. Cloner le dépôt :
   ```bash
   git clone https://github.com/Alioune205/mon-site.git
   ```
2. Ouvrir `index.html` dans un navigateur.
