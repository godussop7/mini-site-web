# Mon Mini Site Web

## Description
Ce projet est un mini site web réalisé dans le cadre d’un TP sur Git et GitHub.

Le site contient :
- Une page HTML
- Un fichier CSS
- Un fichier contenant les réponses aux questions du TP

## Technologies utilisées
- HTML5
- CSS3
- Git
- GitHub
